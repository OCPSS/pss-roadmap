# pss-roadmap
Guia de estudos da disciplina de Projetos de Sistemas de Software com foco no desenvolvimento de sistemas em Java (JDK 17 ou 21) e orientação a objetos. Abrange princípios de engenharia de software como SOLID, baixo acoplamento, alta coesão e padrões de projeto (GoF). Inclui modelagem UML, uso de Maven, Git/GitHub e integração com banco de dados.
